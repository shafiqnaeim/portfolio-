PulseOn — Wind direction & environmental monitoring

Jan 2026

MicroPython · ThingSpeak · Raspberry Pi Pico · GPIO · I2C LCD · DHT22

  • Added an I2C LCD for live direction readouts on site, plus optional DHT22 reads for temperature and humidity when environmental context matters alongside wind.
  • Integrated ThingSpeak so direction (and optional weather) data can be viewed, logged, and monitored remotely—closing the loop from edge device to cloud.
  • Demonstrates a complete embedded narrative on constrained hardware: sensing, decoding, local display, and cloud telemetry suitable for coursework and portfolio demos.
